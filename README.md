# JogoComSocket
Um jogo de tiro _multiplayer local_ com o uso de sockets em _Java_

## Sobre o projeto
Trabalho final da disciplina de redes de computadores do curso de técnico em informática em 2017, desenvolvido em _Java_.  
Nenhuma nova alteração será feita no projeto. Ele serve apenas como **fonte de consulta** de **anti patterns** e como **arquivo pessoal**.

## Anti patterns
Se você consultar o projeto, fique ciente de que o código:
* Está escrito em português
* Está mal estruturado
* Não usa design patterns
* Possui muitos comentários
* Não utiliza as melhores práticas de OO
